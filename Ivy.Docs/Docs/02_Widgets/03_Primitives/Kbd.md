# Kbd

The Kbd widget displays keyboard shortcuts or key combinations with proper styling. It helps users identify key commands and improves documentation clarity.

```csharp demo-below
Layout.Horizontal() | 
    new Kbd("Ctrl + C") | 
    new Kbd("Shift + Ctrl + C")
```
